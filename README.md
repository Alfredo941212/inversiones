# Sistema de Inversiones

Aplicación web desarrollada con Laravel para administrar inversiones, activos y transacciones de los usuarios.

## Objetivo

Facilitar el registro, control y seguimiento de inversiones mediante una plataforma web organizada y segura.

## Funcionalidades

- Registro e inicio de sesión de usuarios.
- Gestión de activos.
- Registro de transacciones.
- Consulta del historial de movimientos.
- Administración de información financiera.
- Protección de rutas.
- Autenticación de usuarios.
- Uso de Laravel Fortify.

## Tecnologías utilizadas

- PHP 8.3
- Laravel
- MySQL
- Laravel Fortify
- Livewire
- HTML
- CSS
- JavaScript
- Git
- GitHub
- Laragon
